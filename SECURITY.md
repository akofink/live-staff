# Security Policy

## Supported Versions

Security fixes are applied to the latest version on the `main` branch.

## Reporting a Vulnerability

Do not report suspected vulnerabilities in a public issue.
Use GitHub's private vulnerability reporting for this repository.
If private reporting is unavailable, contact the project maintainer privately through GitHub with enough detail to reproduce and assess the issue.

Please include the affected version or commit, a clear description, reproduction steps or proof of concept, and potential impact.
